Zen Grids

A responsive grid system built with the power of Sass

Using Zen Grids is easy!

Here’s a simple example, a content column with a sidebar on each side, aligned to a 8 column grid:

@import "zen";



$zen-column-count: 8; // Set the total number of columns in the grid.
$zen-gutter-width: 40px; // Set the gutter size. A half-gutter is used on each side of each column.



.container {
  @include zen-grid-container;
}
.sidebar1 {
  @include zen-grid-item(1, 2); // Apply this mixin for each grid item in the container.
}




.content {
  @include zen-grid-item(3, 5); // Position this grid item in the 3rd column. // Make this grid item span 5 columns.
}

.sidebar2 {
  @include zen-grid-item(8, 1);
}

That’s it! 1 import statement, 2 variables, and 2 mixins. You’re done. And, here’s the results:

Plus, within the .container element, the .sidebar1, .sidebar2 and .content items can be arranged in any HTML source order.

What are Sass and Compass?

Sass = CSS + Awesomeness

Sass is a super-set of CSS. Any valid CSS code is valid Sass. But Sass adds developer-friendly compabilities such as nesting, variables, arithmatic and mixins. And Sass gets compiled into regular CSS so its 100% compatible with all browsers.

Compass is a tool to help build libraries of reusable, pre-made Sass code. You can learn more at:

sass-lang.com
and
compass-style.org

Flow Module: Configurable Variables #

Compass’ reference website has documentation about working with configurable variables.

Common variables #

$zen-auto-include-flow-item-base #

This variable works the same as $zen-auto-include-grid-item-base, except for flow items instead of grid items. You can generate more efficient CSS if you set this to false and manually apply the zen-grid-item-base() mixin to all flow items from within a single ruleset. 

$zen-auto-include-flow-item-base: true !default;

Flow Module: Mix-ins #

zen-grid-flow-item( $column-span, $parent-column-span )
zen-grid-flow-item( $column-span, $parent-column-span, $alpha-gutter, $omega-gutter )
or for fixed length (em, px, etc.) layouts:
zen-grid-flow-item( $column-span )
zen-grid-flow-item( $column-span, $alpha-gutter: [value], $omega-gutter: [value] ) #

To use this mixin, you must first import the flow module with: @import "zen-grids/flow";

Apply this to an HTML item that is in the normal flow of a document to help align it to the grid. Set the $column-span to the number of columns that the HTML element should span. For responsive layouts with a percentage-based grid width, set the $parent-column-span to the number of columns that the parent element spans; fixed-unit layouts using px, em, etc. do not need to specify this.

Unlike the zen-grid-item() mixin, this mixin does not float the HTML item; see the related zen-float() mixin. This mixin also does not have a half-gutter on each side. By default, it has no gutter in the "alpha position" (the left side) and a full gutter in the "omega position" (the right side.) You can turn on or off the alpha and omega gutters by setting the $alpha-gutter and $omega-gutter parameters to true or false.

Note: when the $direction is set to right (for RTL languages), the alpha position is on the right and the omega position is on the left.

Additional optional parameters and their defaults:

  • $direction: $zen-direction
  • $columns: $zen-columns
  • $gutters: $zen-gutters
  • $grid-width: $zen-grid-width
  • $box-sizing: $zen-box-sizing
  • $switch-direction: $zen-switch-direction
  • $auto-include-flow-item-base: $zen-auto-include-flow-item-base

zen-float() #

To use this mixin, you must first import the flow module with: @import "zen-grids/flow";

Apply this to flow items that need to be floated.

Optional parameters and their defaults:

  • $direction: $zen-direction
  • $switch-direction: $zen-switch-direction

Example usage #

In this example layout, we have positioned two grid items (using the zen-grid-item() mixin); one spanning 4 columns and one spanning 1 column. Both grid items have a dotted border. We've also placed a solid red border around the first paragraph in the left grid item. It is in the normal flow along with the second paragraph.

Now we’ve applied @include zen-grid-flow-item(2, 4); to the first paragraph. It now aligns to the first two columns of the grid item. We must tell the mixin that the parent grid item spans 4 items so that the width of the flow item is calculated properly. Note that the first paragraph is still in the normal flow along with the second paragraph.

Now, in addition to the @include zen-grid-flow-item(2, 4);, we’ve also applied @include zen-float(); to the first paragraph. The first paragraph now floats to the left and allows other normal flow items (like the second paragraph) to wrap around it.
